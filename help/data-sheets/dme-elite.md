# 精英支持Creative Cloud和Adobe Document Cloud

![图标](assets/Elitebanner.png)

Adobe 提供多种多样的技术资源以帮助支持您的业务，这些技术资源作为您的 Adobe 企业订阅的一部分包括在内。ELITE支持计划对此进行了改进。 ELITE客户可以访问指定支持工程师和技术客户经理，他们与您合作，在Adobe支持团队中充当您指定的技术联系人时提供一流的主动和被动支持。

借助在适用的Creative Cloud和Document Cloud解决方案方面的丰富经验，您的支持团队可以确保无论您的支持需求多么复杂，Adobe支持团队都会与您一起，确保您在适用的Adobe解决方案方面的投资达到最大限度，并帮助您在问题发生之前避免问题。 您还可以利用我们详细且深入的产品技术文档和最新发行说明。
