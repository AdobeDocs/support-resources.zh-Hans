# Adobe Admin Console中的辅助功能

适用于企业和团队。

了解[!DNL Adobe Admin Console]中的辅助功能支持，该支持提供了一个中央位置，用于管理整个组织中的Adobe授权。

## 操作系统无障碍支持

许多广泛使用的操作系统的辅助功能可用于[!DNL Admin Console]。 这些辅助功能包括高对比度和其他显示设置。 但是，由于[!DNL Admin Console]的当前用户界面，可能存在一些限制。

## 键盘支持

[!DNL Admin Console]上的大多数交互式内容都可以单独使用键盘导航和操作。 例如，您可以将&#x200B;**[!UICONTROL Tab]**&#x200B;或&#x200B;**[!UICONTROL Shift]** + **[!UICONTROL Tab]**&#x200B;用于导航和控制。 导航到正确的链接或按钮后，按&#x200B;**[!UICONTROL Enter]**&#x200B;或&#x200B;**[!UICONTROL Return]**&#x200B;键将其选定。 仍可能会存在一些例外，如[无障碍合规性报告](https://www.adobe.com/accessibility/compliance.html)中所述（_仅英文_）。

## 辅助技术支持

[!DNL Admin Console]为辅助技术（包括屏幕阅读器和屏幕放大镜）提供了一些支持。 有关详细信息，请参阅[无障碍合规性报告](https://www.adobe.com/accessibility/compliance.html)。

## Admin Console支持哪些辅助功能？

[[!DNL Admin Console]](https://adminconsole.adobe.com/)提供了一个中央位置，用于管理整个组织中的Adobe授权。 它提供辅助型技术支持、键盘支持和操作系统无障碍支持（有一些例外）。

## 有反馈吗？

[!DNL Admin Console]团队正在不断努力改进其辅助功能和用户体验。 您可以在[企业和团队社区](https://community.adobe.com/enterprise-teams-622)中分享您对辅助功能的反馈。
