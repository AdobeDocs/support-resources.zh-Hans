# 在Admin Console中强制实施密码过期

> **适用于：**&#x200B;企业和团队

组织系统管理员可以按照该过程为其用户设置密码过期时间限制。

## 环境

[Adobe Admin Console](https://adminconsole.adobe.com/)

## 步骤

1. 转到[Adobe Admin Console](https://adminconsole.adobe.com/settings) **[!UICONTROL 设置]**&#x200B;部分并展开左上角的&#x200B;**[!UICONTROL 隐私和安全性]**&#x200B;选项。

1. 选择&#x200B;**[!UICONTROL 身份验证设置]**。

1. 查看&#x200B;**[!UICONTROL 身份验证级别]**&#x200B;并为您的用户选择最合适的身份验证级别： **[!UICONTROL 对用户来说最简单]**，**[!UICONTROL 更安全]**，**[!UICONTROL 最安全]**。

访问[身份验证设置Helpx](https://helpx.adobe.com/enterprise/using/authentication-settings.html)，了解有关可用身份验证和安全策略的更多信息。
