# Adobe Workfront支持概述

Adobe Workfront将加入Adobe Admin Console Platform。

在未来几个月中，将联系客户以协调对Adobe Identity和Admin Console的迁移，这是一个更安全、功能更完善的系统管理控制台。 以下为即将进行的更改带来的好处：

* 使用单个Workfront身份对Adobe和其他产品进行身份验证。
* 在生产环境和沙盒环境或预览环境之间单独管理用户权限。
* 为Workfront用户利用多个SSO配置。
* 使用与其他Workfront产品相同的方式管理对Adobe的权限。

若要深入查看Adobe标识和Admin Console，您可以引用此[支持页面](https://helpx.adobe.com/cn/enterprise/admin-guide.html)。 此外，还将当前Workfront控制台与新Adobe Admin Console进行比较[此处](https://one.workfront.com/s/document-item?bundleId=the-new-workfront-experience&topicId=Content%2FAdministration_and_Setup%2FGet_started-WF_administration%2Factions-in-admin-console.htm&_LANG=enus)。

## 客户迁移详细信息

与现有客户联系，以安排从2022年7月开始进行的迁移。 我们的迁移团队支持同事指导客户完成该过程，提供有关控制台设置的建议，并提供所需的文档链接以使迁移尽可能简单和轻松。

自2022年7月起，新增客户净额自动计入Adobe Admin Console，无需任何进一步行动。
